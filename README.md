## loopback-connector-firebase

Firebase connector for loopback

Documentation: TBD


## Customizing Firebase configuration for tests/examples

The .loopbackrc file is in JSON format, for example:

    {
        "test": {
            "firebase": {
                "url": "<firebase_url>",
                "token": "<security_token>"
            }
        }
    }
